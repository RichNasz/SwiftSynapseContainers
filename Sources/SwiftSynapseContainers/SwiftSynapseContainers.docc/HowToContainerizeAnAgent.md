# How to Containerize an Agent

Step-by-step guide to adding `@Containerized` to an existing `@SpecDrivenAgent` actor.

## Overview

Containerizing an agent is an additive change — you add one macro and one method call. No changes to `execute(goal:)`, no changes to tool registration, no changes to the Harness configuration.

## Before You Start

Confirm you have:
- `SwiftSynapseContainers` in your `Package.swift` (replaces `SwiftSynapseHarness`)
- The `container` CLI installed: `brew install container`
- The target image pulled: `container pull swift:latest`
- The `com.apple.security.virtualization` entitlement for distribution builds

## Step 1: Add @Containerized

Place `@Containerized` directly above `@SpecDrivenAgent`:

```swift
// Before:
@SpecDrivenAgent
public actor AnalysisAgent { ... }

// After:
@Containerized(image: "swift:latest", policy: .strict)
@SpecDrivenAgent
public actor AnalysisAgent { ... }
```

The macro generates three members. You don't write these — they appear automatically:

```swift
// Generated by @Containerized:
public var containerConfig: ContainerConfiguration = ContainerConfiguration(
    image: "swift:latest",
    cpuCount: 2,
    memoryGB: 4,
    diskGB: 20,
    policy: .strict,
    networkEnabled: false,
    mounts: [],
    environment: [:],
    timeoutSeconds: 300
)
public var _containerID: String? = nil
public func containerized(goal: String) async throws -> ContainerizedResult<String> { ... }
```

## Step 2: Choose the Right Policy

Match ``SandboxPolicy`` to your trust model:

```swift
// Untrusted code — no network, no mounts (safest)
@Containerized(image: "swift:latest", policy: .strict)

// Trusted agent that calls an LLM API
@Containerized(image: "swift:latest", policy: .standard, timeoutSeconds: 120)

// Trusted agent that needs to read input files
@Containerized(image: "ubuntu:24.04", policy: .permissive)
// Then add mounts to containerConfig after init if needed
```

## Step 3: Size the VM

Set `cpuCount`, `memoryGB`, and `diskGB` based on your workload. The defaults (2 vCPU, 4 GB, 20 GB) suit most LLM-calling agents:

```swift
@Containerized(
    image: "swift:latest",
    cpuCount: 4,       // increase for compute-heavy tasks
    memoryGB: 8,       // increase if the agent loads large models or datasets
    diskGB: 40,        // increase for agents that write large files
    timeoutSeconds: 600
)
```

Use `ContainerizedResult.peakMemoryBytes` from a test run to right-size `memoryGB`.

## Step 4: Call containerized(goal:)

Switch the call site from `run(goal:)` to `containerized(goal:)` for workloads that need isolation:

```swift
// Harness run — hooks, permissions, and resilience apply, execute() runs in host:
let response = try await agent.run(goal: goal)

// Containerized run — execute() runs inside the MicroVM:
let result = try await agent.containerized(goal: goal)
let output = result.value
```

You can use both on the same actor — `run(goal:)` for trusted goals, `containerized(goal:)` for untrusted ones.

## Step 5: Handle Container Errors

Add a catch block for ``ContainerizedAgentError`` alongside your existing error handling:

```swift
do {
    let result = try await agent.containerized(goal: userInput)
    return result.value
} catch ContainerizedAgentError.imageNotFound(let image) {
    throw MyError.missingImage(image)
} catch ContainerizedAgentError.timeoutExceeded(let seconds) {
    throw MyError.agentTimeout(seconds)
} catch ContainerizedAgentError.containerToolNotFound {
    throw MyError.setupRequired("Install container CLI: brew install container")
}
```

## Complete Example

```swift
import SwiftSynapseContainers

@Containerized(image: "swift:latest", cpuCount: 2, memoryGB: 4, policy: .strict)
@SpecDrivenAgent
public actor CodeReviewAgent {
    private let config: AgentConfiguration

    public init(configuration: AgentConfiguration) throws {
        self.config = configuration
        _ = try configuration.buildLLMClient()
    }

    public func execute(goal: String) async throws -> String {
        let client = try config.buildLLMClient()
        let agent = Agent(client: client, model: config.modelName)
        return try await agent.respond(to: goal)
    }
}

// Usage:
let agent = try CodeReviewAgent(configuration: config)
let result = try await agent.containerized(goal: "Review this PR for security issues:\n\(prDiff)")
print(result.value)
print("Ran in \(result.wallTimeSeconds)s using \(result.peakMemoryBytes / 1_048_576) MB")
```
